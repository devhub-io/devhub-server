# Devhub.io API

> Using Egg.js

## Build Setup

```bash
# install dependencies
$ npm install

# env config
$ cp .env.js.example .env.js

# serve with hot reload at localhost:7001
$ npm run dev

# build for production and launch server
$ npm run start

# Lint & Test
$ npm run ci
```
